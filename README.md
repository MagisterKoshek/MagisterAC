# MagisterAC

Advanced Minecraft AntiCheat for Paper, Purpur, Folia, and Spigot servers.

## Features

* 35+ cheat detection checks
* Native Anti-XRay protection
* Replay system
* Staff observation tools
* Freeze system
* Watchlist
* Discord integration
* Interactive GUI panels
* Case management

## Supported Platforms

* Paper
* Purpur
* Folia
* Spigot

## Requirements

* Minecraft 1.21.x
* Java 21+

## Installation

1. Download the latest release.
2. Place `MagisterAC.jar` into your `plugins` folder.
3. (Optional) Install ProtocolLib.
4. Start the server.
5. Configure the plugin using the generated configuration files.
